2) 메소드(method) 만들기
- 메소드 = 함수(function): 나홀로 존재
    : 클래스 안에 선언되어 있어야 메소드
    : 혼자 존재할 수 있으면 함수

    : 반복적으로 사용되는 기능을 하나의 단위로 만든 것.

    - 생성 : 기능을 만듦.
    - 호출 : 기능을 가용.

    - 반환할 값이 없으면 void이고, void 타입의 메소드일 경우 return; 해도 되고, return문 자체를 생략가능

    - 메소드를 호출하면 그 메소드가 있는 곳으로 이동한다.
      메소드 내부 기능을 실행하고 return하면 메소드를 호출했던 위치로 되돌아옴.

    - 메소드명을 이용해서 호출하고, 필요한 경우 값을 전달 (호출하는 쪽에서는 argument, 생성하는 쪽은 parameter라고 함.)

    - static 메소드는 non static에서는 직접 호출 불가능
    --> 반드시 생성(new)을 통해서 호출할 수 있다.

뭘 생성하나? 사용하고자하는 메서드를 포함하는 클래스를 생성해야됨

클래스명 참조변수명 = new 클래스명();
Scanner sc = new Scanner(System.in);
MethodTest_03 mt = new MethodTest_03();

mt를 통해서만 non static 메소드에 접근할 수 있다.


  <생성하는 방법>
  [접근지정자] [키워드-static, final] [반환타입] 메소드명(파라미터){
    // 실행문;
  }

  // method signiture
  public static void main(String[] args) {

  }

  // 매개변수(parameter) 입력값, 지역변수임(호출하면서 초기화 함.)
  public static double multiply(int a, int b) {
    double result = a * b;
    return result;
  }
