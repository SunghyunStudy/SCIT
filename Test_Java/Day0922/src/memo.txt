abstract class -
    상속 전용 클래스
    멤버변수, 멤버메소드, 생성자
    하위클래스에서는 상위 클래스에서 상속해준 추상메소드가 있다면 반드시 오버라이딩해야 객체를 생성할 수 있다.

interface
    객체 생성을 못하는 abstract
    일반메소드는 가질 수 없다.
    추상메소드와 디폴트매소드와 스태틱메소드만 가질 수 있다.
    멤버변수, 생성자를 가질 수 없다.
    변수의 위치에 있는 변수는 전부 상수다. = static final이 붙음.
    다중 상속을 할 수 있도록 하기 위해 만든 개념이기도 함.

    프로젝트를 할 때 구현해야되는 클래스들의 가이드를 하기 위해 만드는 것임.

    다형성과 동적바인딩을 위해서 만들어짐...
