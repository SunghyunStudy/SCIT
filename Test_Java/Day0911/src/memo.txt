변수(Variable)
1) 지역변수 (Local Variable)
    - 블럭 내에서 선언된 변수
    - 초기화 후 사용
    - 메소드 내부에서 선언된 변수
    - 블럭이 닫히면 사용 불가능
    - Stack 메모리 공간에 저장

2) 멤버변수 (Member Variable)
    - 인스턴스 변수
    - 초기화를 안하면 자동 초기화(0, 0.0, false)
    - 멤버변수를 사용하려면 클래스를 인스턴스화(생성)한 후에 사용가능 (= 객체로 사용)
    - 객체(멤버변수)는 생성하면 메모리에 올라가고
      그 객체를 아무도 참조하지 않으면 사라짐.

3) 정적변수 (Static variable)
    - 멤버변수가 선언되는 위치에 선언됨
    - static이 붙은 변수는 생성없이 사용가능하다
    - 클래스변수라고 부름 (딱 한번만 메모리에 올라감)
    - 여기저기에서 공유해서 사용하고 싶을 때 사용.
    - 프로그램이 구동되자마자 가장 먼저 메모리에 올라감.
    - 프로그램 종료되면 사라짐.

4) 상수변수 (Constant Variable)
    - 클래스변수 앞에 final을 붙인 거.
    - 변수명을 모두 대문자로만 사용
    - 두 개의 단어가 필요하면 _(언더스코어)로 사용

    static final int MY_DATA = 10;

5) final 변수
    - final 키워드를 변수 선언할 때 앞에 붙여서 사용.
    - 선언과 동시에 초기화 해야됨.
    - 값을 변경할 수 없음


** 반복문 ( for, while, do-while )
1) 조건식을 먼저 판단하기 떄문에 조건이 처음부터 false이면, 한번도 반복하지 못할 수 있다. = while, for

2) 조건식을 먼저 판단하기 떄문에 조건이 처음부터 false여도, 적어도 한번 실행 가능. = do-while

초기식;
while(조건식) {
    실행문;
    증감식;
}

    문장(3);
}

초기식;
for(;;){
    조건식;
    문장;
    증감식;

}

--- 배열(array) => 1, 2차원
- 같은 데이터 타입의 변수가 여러개 필요할 때
- 배열의 생성 : new 라는 연산자를 이용해서 만든다.

    int a, b, c, d, e;

    - 배열의 index는 0부터
    - 메모리 공간에 연속적으로 생성이 됨.
    - 변수 a는 참조변수라서 주소값이 들어감.
    - 배열의 값은 기본값으로 초기화 됨. [0, 0, 0, 0, 0]

    int[] a = new int[5];                       // 생성
    int[] b = new int[5] {1, 2, 3, 4, 5};       // 생성 + 초기화 / 선언과 초기화를 분리시킬 수 있음.

    int[] b;

    int[] c;
