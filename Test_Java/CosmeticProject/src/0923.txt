자료구조(Data Structure)

1) 선형 (Linear)
    - 배열 : CRUD
        - 배열에 순차적으로 쌓음. : 데이터를 찾을 때 처음부터 끝까지 다 찾아야됨(Full Scan방식)
        - 삭제를 하게 되면 뒤의 요소들을 앞으로 당겨야 됨.
        - Runtime의 크기를 늘릴 수 없다.

    - ArrayList
        - Runtime 크기를 알아서 늘려줌
        - 순차적으로 쌓음. 데이터를 꺼낼 때 마음대로 꺼낼 수 있음

    - LinkedList
        - 위치 값을 갖고 순차적으로
        - 앞에서부터 순차적으로 찾을 수 있음. 중간에 빈 값을 끼워넣기에 용이. 빼는 것도 굳이 데이터를 당길 필요가 없어짐.


    - Stack (스택) : 선입후출구조.
        ※ 자바의 지역변수가 스택에 쌓임)
        ※ System Call 에서 사용. (재귀적 함수를 말하는 듯)


    - Queue(큐) : 선입선출 - (인쇄대기열)
        ※
